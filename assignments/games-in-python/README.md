
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. You will create a playable Hangman game that hides a word, tracks guesses, and ends when the player wins or runs out of attempts.

## 📝 Tasks

### 🛠️ Select and reveal the hidden word

#### Description
Create the core word-selection and display logic for the game.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Show the hidden word using underscores for unguessed letters
- Update the display as correct letters are guessed

### 🛠️ Track guesses and finish the game

#### Description
Build the input loop that handles guesses, tracks mistakes, and ends the game.

#### Requirements
Completed program should:

- Accept single-letter guesses from the player
- Track incorrect guesses remaining
- End when the word is fully guessed or attempts are exhausted
- Display clear win and lose messages
